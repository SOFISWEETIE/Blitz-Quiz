# Blitz Quiz

**Blitz Quiz** es una aplicación web diseñada como un juego de preguntas donde el usuario debe responder antes de que el tiempo se agote  
Se trata de una **Single Page Application (SPA)** desarrollada con **Angular 20.3.15**, que integra **autenticación de usuarios con Google**, **control del tiempo**, y un **sistema de puntuación 
y ranking en tiempo real**, para ofrecer una experiencia **intuitiva, dinámica y accesible desde cualquier dispositivo**


## Tecnologías utilizadas

- **Angular** (Standalone Components)
- **TypeScript**
- **Firebase**
  - Firebase Authentication
- **HTML5**
- **CSS3**
- **Node.js**
- **npm**


## Requisitos previos

Para poder ejecutar el proyecto en un entorno local es necesario los siguientes elementos:

- Node.js (versión LTS recomendada)
- Angular CLI
- Navegador web moderno (Chrome, Firefox, Edge, etc.)


##  Manual de instalación y ejecución

Para iniciar el proyecto en local, hay que seguir los siguientes pasos:

1. **Clonar el repositorio y acceder a la carpeta del proyecto**
```bash
   git clone https://github.com/SOFISWEETIE/Blitz-Quiz.git
   cd Blitz-Quiz
```  

2. **Instalar las dependencias necesarias**
```bash
npm install
```  


3. **Levantar la aplicación en local**
```bash
ng serve
```  


4. **Acceder a la aplicación desde el navegador**
```text
http://localhost:4200
```
